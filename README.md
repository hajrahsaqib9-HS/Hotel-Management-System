# Hotel Management System (Python)

## Description
This is a Data Structure project implemented in Python using:
- Arrays (Lists)
- Stack
- Queue

## Features
- Room Booking
- Room Cancellation
- Waiting List Management
- Cancelled Booking History

## Data Structures Used
- Array (Rooms)
- Stack (Cancelled bookings)
- Queue (Waiting list)

## How to Run
1. Install Python
2. Run the file:
   python hotel_management_system.py
